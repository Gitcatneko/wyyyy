# wangyiyun_app
后端接口使用网易云音乐node.js离线版
需要下载网易云音乐离线接口NeteaseCloudMusicApi
https://binaryify.github.io/NeteaseCloudMusicApi/
然后在src/request/api目录下修改index.js内的baseURL
## Project setup
```
npm install
```
